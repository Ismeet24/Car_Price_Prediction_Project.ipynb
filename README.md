# 🚗 Car Price Prediction using Machine Learning

This project focuses on predicting the price of cars using Machine Learning techniques. It includes data preprocessing, feature engineering, model training, and evaluation.

---

## 📌 Project Overview

Car price depends on multiple factors such as:
- Brand value
- Mileage
- Engine power
- Age of the car

This project builds a predictive model to estimate car prices accurately.

---

## ⚙️ Technologies Used

- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Google Colab

---

## 🔍 Workflow

1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Exploratory Data Analysis
5. Model Training
   - Linear Regression
   - Random Forest Regressor
6. Model Evaluation
7. Feature Importance Visualization
8. Prediction System

---

## 📊 Model Performance

- Random Forest performed better than Linear Regression
- Evaluation Metrics:
  - R² Score
  - Mean Absolute Error (MAE)
